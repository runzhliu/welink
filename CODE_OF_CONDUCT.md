# 行为准则

## 我们的承诺

为了营造一个开放、友好、多元、包容、健康的社区，我们作为贡献者和维护者承诺：让每个人参与本项目都能获得无骚扰的体验——无论年龄、体型、可见或不可见的残障、民族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教、性取向。

## 我们的标准

**积极行为的例子**：

- 表达善意和共情
- 尊重不同的观点和经历
- 给予和优雅地接受建设性反馈
- 为错误承担责任并向受影响的人道歉
- 关注对社区整体最好的做法，而不仅是个人利益

**不可接受的行为**：

- 使用性化语言或图像，以及任何形式的性骚扰
- 挑衅、侮辱、贬损的评论和人身攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息（物理或电子地址等）
- 其他在专业场合不当的行为

## 执行

维护者有责任澄清和执行我们对可接受行为的标准，对他们认为不当、威胁、冒犯或有害的任何行为采取适当公正的纠正措施。

维护者有权移除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、issue 和其他贡献，并在适当时解释调节决定的原因。

## 范围

本行为准则适用于所有社区空间（包括 GitHub、Issues、PR、Discussions），以及个人在公开场合正式代表社区时。

## 举报
